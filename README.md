🛍️ Retail Customer Behavior Trends Analysis
📌 Overview

This project presents a complete end-to-end Data Analytics workflow focused on analyzing retail customer behavior trends. The objective was to extract meaningful business insights from raw transactional data and transform them into actionable recommendations through structured analysis and interactive visualization.

The project demonstrates practical experience in data cleaning, exploratory data analysis (EDA), SQL-based querying across multiple database systems, and business intelligence dashboard development.

📂 Dataset

The dataset contains retail transactional data, including:

Customer details

Product information

Sales transactions

Purchase frequency

Revenue and category-level performance

The data required preprocessing due to:

Missing values

Inconsistent formatting

Duplicate records

Outliers

🛠 Tools & Technologies

Programming & Data Processing

Python (Pandas, NumPy, Matplotlib, Seaborn)

Database Systems

PostgreSQL

MySQL

SQL Server

Query Language

SQL (Joins, Aggregations, Window Functions, Subqueries, CTEs)

Business Intelligence

Power BI
